# 临床运动处方实践专家共识（2025）子技能

## 技能概述
**名称**：临床运动处方专家
**上级技能**：运动康复小南
**依据标准**：《临床运动处方实践专家共识（2025）》
**适用场景**：医疗机构、康复中心、社区健康服务中的运动处方制定与实施

---

## 核心流程

### 一、运动处方基本流程

遵循"评估→诊断→处方→实施→监测→调整"六步闭环流程：

```
┌─────────────┐
│  1. 评估    │ → 健康风险评估 + 体适能评估
├─────────────┤
│  2. 诊断    │ → 运动能力分级 + 受限因素识别
├─────────────┤
│  3. 处方    │ → FITT-VP 原则 + 个体化调整
├─────────────┤
│  4. 实施    │ → 运动环境 + 过程监测
├─────────────┤
│  5. 监测    │ → 不良反应识别 + 处理
├─────────────┤
│  6. 调整    │ → 定期复评 + 处方进阶
└─────────────┘
```

## 健康风险评估

### 1. PAR-Q+ 问卷筛查
运动前健康筛查的 7 个核心问题：
- 心脏疾病史
- 胸痛症状
- 晕厥或平衡障碍
- 骨骼关节问题
- 降压药物使用
- 其他健康问题

### 2. 心血管风险分层

| 风险等级 | 标准 | 运动建议 |
|---------|------|---------|
| **低危** | 无症状且≤1 个心血管危险因素 | 可自行开始中等强度运动 |
| **中危** | ≥2 个心血管危险因素或稳定型慢性病 | 建议医学评估后开始运动 |
| **高危** | 心血管疾病、近期事件、不稳定病情 | 需医学评估和运动负荷试验 |

### 3. 运动禁忌症

#### 绝对禁忌症
- 不稳定型心绞痛
- 急性心肌梗死（72 小时内）
- 未控制的心律失常
- 急性心力衰竭
- 严重主动脉瓣狭窄
- 急性肺栓塞或肺梗死
- 急性心肌炎或心包炎
- 急性感染或发热
- 未控制的严重高血压（≥180/110mmHg）

#### 相对禁忌症
- 左心室流出道梗阻
- 未控制的代谢性疾病
- 未控制的精神疾病
- 严重贫血
- 电解质紊乱
- 妊娠合并症

## 体适能评估

### 评估项目与方法

| 评估维度 | 推荐方法 | 适用人群 |
|---------|---------|---------|
| **心肺耐力** | 6 分钟步行试验 (6MWT) | 老年、体弱、慢病患者 |
| | 心肺运动试验 (CPET) | 心血管患者、运动员 |
| | 台阶试验 | 一般人群 |
| **肌肉力量** | 握力测试 | 所有人群 |
| | 1RM 测试 | 无禁忌证者 |
| | 30 秒坐站测试 | 老年人 |
| **柔韧性** | 坐位体前屈 | 一般人群 |
| | 关节活动度测量 | 特定关节评估 |
| **平衡能力** | Berg 平衡量表 | 老年人、神经系统疾病 |
| | 单脚站立测试 | 一般人群 |
| | TUG 测试 | 老年人 |
| **身体成分** | BMI、腰围 | 所有人群 |
| | 体脂率 | 所有人群 |

## 疾病特异性运动处方

### 一、心血管疾病

#### （一）高血压

**评估要点**：
- ASCVD 风险评估
- 靶器官损害评估
- 血压≥180/110mmHg 暂缓运动

**运动处方**：
| 运动类型 | 频率 | 强度 | 时间 | 注意事项 |
|---------|------|------|------|---------|
| 有氧运动 | ≥5 天/周 | 中等 (40-60% HRR) | 30-60 分钟/天 | 避免憋气 |
| 抗阻运动 | 2-3 天/周 | 轻 - 中 (30-50% 1RM) | 8-10 个动作 | 避免等长收缩 |
| 柔韧平衡 | 2-3 天/周 | 轻柔 | 10-15 分钟 | - |

**特别警示**：
- 避免 Valsalva 动作（憋气用力）
- 注意降压药对运动反应的影响
- 运动后不要突然停止

#### （二）冠心病

**运动处方**：
- **有氧运动**：中等强度（40-70% HRR 或 RPE 11-14），每周 3-7 天
- **抗阻运动**：中等强度（50-70% 1RM），每周 2-3 天
- **开始时机**：
  - PCI 术后：1-4 周开始
  - CABG 术后：4-8 周开始

#### （三）心力衰竭

**运动处方**：
- **有氧运动**：低 - 中等强度（40-70% 峰值心率），从 10-15 分钟开始渐进
- **抗阻运动**：轻强度（30-50% 1RM），避免上肢过度用力
- **IMT 训练**：30% 最大吸气压，每天 15 分钟

### 二、代谢性疾病

#### （一）2 型糖尿病

**评估要点**：
- 血糖控制情况
- 糖尿病并发症筛查（视网膜、肾脏、神经、足部）
- 心血管风险评估

**运动处方**：
| 运动类型 | 频率 | 强度 | 时间 | 注意事项 |
|---------|------|------|------|---------|
| 有氧运动 | ≥5 天/周 | 中等 (50-70% HRR) | 累计≥150 分钟/周 | 防低血糖 |
| 抗阻运动 | 2-3 天/周 | 中等 (50-70% 1RM) | 8-10 个动作 | 避免胰岛素注射部位 |
| 柔韧平衡 | ≥2 天/周 | 轻柔 | 10-15 分钟 | 注意足部保护 |

**血糖管理**：
- 运动前血糖目标：5.6-13.9 mmol/L
- 运动前后血糖监测
- 随身携带快速升糖食物
- 注意足部保护

#### （二）骨质疏松

**运动处方**：
- **负重有氧**：快走、慢跑、跳舞，每周 3-5 天
- **抗阻运动**：中等强度（60-80% 1RM），重点训练髋部和脊柱伸肌
- **冲击性运动**：低至中等冲击（如跳绳、踏步），骨折高风险者慎用
- **平衡训练**：每天进行，预防跌倒

**特别警示**：
- 避免脊柱屈曲和旋转动作（如仰卧起坐）
- 骨折高风险者避免高冲击运动

### 三、呼吸系统疾病

#### 慢阻肺（COPD）

**评估要点**：
- 肺功能分级（GOLD 分级）
- 症状评估（mMRC、CAT）
- 急性加重史

**运动处方**：
| 运动类型 | 频率 | 强度 | 时间 | 注意事项 |
|---------|------|------|------|---------|
| 有氧运动 | 3-5 天/周 | 中等 (60-80% 峰值心率) | 20-60 分钟 | SpO2≥88% |
| 抗阻运动 | 2-3 天/周 | 中等 (60-80% 1RM) | 8-10 个动作 | 避免憋气 |
| 呼吸肌训练 | 每天 | 30-50% 最大吸气压 | 15-20 分钟 | 缩唇呼吸 |

**呼吸技术**：
- 缩唇呼吸
- 腹式呼吸
- 必要时氧疗支持

### 四、神经系统疾病

#### 脑卒中

**开始时机**：发病后 1-3 个月（病情稳定后）

**运动处方**：
| 运动类型 | 频率 | 强度 | 时间 | 注意事项 |
|---------|------|------|------|---------|
| 有氧运动 | 3-5 天/周 | 中等 (RPE 11-14) | 20-60 分钟 | 监测血压 |
| 抗阻运动 | 2-3 天/周 | 中等 (50-70% 1RM) | 8-10 个动作 | 注意患侧 |
| 平衡步态 | ≥3 天/周 | - | 20-30 分钟 | 防跌倒 |
| 柔韧性 | 每天 | - | 10-15 分钟 | 牵伸痉挛肌群 |

### 五、骨骼肌肉疾病

#### 骨关节炎

**运动处方**：
- **有氧运动**：低冲击运动（游泳、骑行、椭圆机），每周≥3 天
- **抗阻运动**：轻 - 中等强度，重点训练关节周围肌肉
- **关节活动度**：每天进行

**特别警示**：
- 避免高冲击、过度负重活动
- 体重管理很重要

## 特殊人群运动处方

### 一、老年人（≥65 岁）

**评估重点**：
- 衰弱筛查（Fried 表型、临床衰弱量表）
- 跌倒风险评估
- 认知功能评估
- 日常生活能力（ADL）

**运动处方**：
| 运动类型 | 频率 | 强度 | 时间 |
|---------|------|------|------|
| 有氧运动 | 3-5 天/周 | 中等 | 从 10 分钟开始渐进 |
| 抗阻运动 | 2-3 天/周 | 中等 | 8-10 个动作 |
| 平衡训练 | ≥3 天/周 | - | 单脚站立、太极 |
| 柔韧性 | ≥2 天/周 | - | - |

### 二、孕产妇

**运动处方**：
- **有氧运动**：中等强度（RPE 12-14），每周≥5 天，20-30 分钟/天
- **抗阻运动**：轻 - 中等强度，每周 2-3 天
- **盆底肌训练**：每天进行

**禁忌**：
- 接触性运动
- 高跌倒风险运动
- 仰卧位运动（孕中期后）

### 三、儿童青少年

**运动处方**：
- **有氧运动**：中 - 高强度，每天≥60 分钟
- **抗阻运动**：中等强度，每周 3 天
- **骨骼强化**：每周≥3 天（跳跃、跑步）

## 运动强度评估方法

### 一、心率法

**最大心率估算**：
- 传统公式：220 - 年龄
- 更新公式：207 - 0.7×年龄

**心率储备法（Karvonen）**：
```
靶心率 = （最大心率 - 静息心率）× 强度% + 静息心率
```

**强度分级**：
| 强度等级 | %HRR | 占最大心率% |
|---------|------|-----------|
| 极低 | <30% | <57% |
| 低 | 30-39% | 57-63% |
| 中等 | 40-59% | 64-76% |
| 高 | 60-89% | 77-95% |
| 极高 | ≥90% | ≥96% |

### 二、主观疲劳度（RPE）

**Borg 6-20 级**：
| RPE | 感觉 | 强度 |
|-----|------|------|
| 9 | 很轻 | 低 |
| 11 | 轻 | 低 |
| 13 | 有点累 | 中等 |
| 15 | 累 | 高 |
| 17 | 很累 | 高 |
| 19 | 很累 | 极高 |

**CR10 级**：
| CR10 | 强度 |
|------|------|
| 2-3 | 中等 |
| 4-6 | 高 |
| 7-8 | 很高 |
| 9-10 | 极限 |

### 三、谈话测试

| 强度 | 表现 |
|------|------|
| 中等 | 可以说话但不能唱歌 |
| 高 | 只能说几个词 |

### 四、代谢当量（METs）

1 MET = 静息代谢率（3.5 ml/kg/min）

| 强度 | METs |
|------|------|
| 低 | <3 |
| 中等 | 3-6 |
| 高 | >6 |

## 运动风险监测

### 一、需停止运动的指征

- 胸痛或胸闷加重
- 明显呼吸困难
- 头晕或晕厥
- 面色苍白、出冷汗
- 收缩压下降≥10mmHg
- 严重心律失常
- SpO2 下降≥4% 或<88%

### 二、实验室指标监测

| 指标 | 安全范围 | 运动建议 |
|------|---------|---------|
| 血红蛋白 | >100g/L | 正常运动 |
| 血红蛋白 | 80-100g/L | 降低强度 |
| 血红蛋白 | <80g/L | 暂停运动 |
| 血小板 | >100×10⁹/L | 正常运动 |
| 血小板 | 50-100×10⁹/L | 低强度，避免碰撞 |
| 血小板 | <50×10⁹/L | 暂停运动 |

## 质量控制

### 处方质量评估指标
- 处方完整性（FITT-VP 要素齐全）
- 处方个体化程度
- 安全性评估充分性
- 处方可行性
- 患者理解度

### 效果评估指标
- 体适能改善率
- 临床指标改善（血压、血糖、血脂等）
- 生活质量评分变化
- 运动依从性
- 不良事件发生率

### 随访计划
| 时期 | 频率 | 内容 |
|------|------|------|
| 初期 | 每周 | 前 4 周，监测适应性 |
| 中期 | 每 2-4 周 | 评估进展，调整处方 |
| 长期 | 每 3-6 个月 | 全面复评，制定新目标 |
