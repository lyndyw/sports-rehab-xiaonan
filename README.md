# 运动康复小南

> 专业运动康复 AI 助手 - 科学康复、重拾活力

## 📖 简介

**运动康复小南** 是一位专业的运动康复 AI 助手，具备运动康复学、物理治疗、运动医学等专业背景。她擅长为患者和运动爱好者提供科学、安全的康复训练指导。

## ✨ 核心功能

| 功能 | 描述 |
|------|------|
| 🏥 康复评估 | 分析运动损伤情况，提供康复建议 |
| 💪 训练指导 | 设计个性化康复训练方案 |
| 📋 运动处方 | 针对各类人群制定运动计划 |
| 🧘 体态纠正 | 改善不良体态和运动模式 |
| 📊 进度追踪 | 监控康复进展，调整训练方案 |

## 🎯 适用场景

- 运动损伤康复（扭伤、拉伤等）
- 术后康复训练
- 慢性疼痛管理（颈肩腰腿痛）
- 体态矫正（脊柱侧弯、圆肩驼背）
- 老年人运动指导
- 青少年体态发育

## 🚀 快速使用

### 方式一：直接复制使用

1. 下载本仓库中的 `运动康复小南/` 目录
2. 将文件复制到你的 AI 助手配置目录
3. 重启应用即可使用

### 方式二：作为 Claude Code Skill 使用

```bash
# macOS/Linux
curl -fsSL https://raw.githubusercontent.com/lyndyw/sports-rehab-xiaonan/main/install.sh | bash

# Windows (PowerShell)
iwr -useb https://raw.githubusercontent.com/lyndyw/sports-rehab-xiaonan/main/install.ps1 | iex
```
